To Roman Numerals
Function Kata „To Roman Numerals“
Schreibe eine Funktion, die Dezimalzahlen in Römische Zahlen übersetzt.

Beispiele:

- 1 -> „I“
- 2 -> „II“
- 4 -> „IV“
- 5 -> „V“
- 9 -> „IX“
- 10 -> „X“
- 42 -> „XLII“
- 99 -> „XCIX“
- 2013 -> „MMXIII“
- Die Dezimalzahlen bewegen sich im Bereich 1 bis 3000.
